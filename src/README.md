# 🛡️<strong> KeenKeeper </strong>

KeenKeeper is a simple and helpful web app. it helps you track your conversations with your friends. You can see how often you call, text, or meet them.

---

## 🚀 Live Link
You can see the project here: [KeenKeeper Demo](https://assignment-7-keenkeeper-hedayet-ullah.netlify.app)

---

## 📝 About the Project
This project is a personal relationship manager. It helps you remember the last time you talked to a friend. It uses charts and lists to show your history.

---

## 🛠️ Technologies I Used
I used these tools to build this project:
* **React.js** - To build the website.
* **Tailwind CSS** - To make it look beautiful and clean.
* **React Router** - To move between different pages.
* **Recharts** - To create the colorful Pie Chart.
* **Context API** - To manage the data in one place.

---

## ✨ Key Features

### 1. Timeline Filter
You can filter your history. You can choose to see only **Calls**, only **Texts**, or only **Videos**. It helps you find information quickly.

### 2. Activity Chart
There is a beautiful **Donut Chart**. It automatically counts your total calls, texts, and videos and shows them in a graph.

### 3. Smart Sorting
The app always shows the **newest action first**. You don't have to scroll down to find your latest conversation; it is always at the top!
